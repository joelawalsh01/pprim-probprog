# P-Prim Terminology Analysis

## The Core Distinction
P-prims (diSessa 1993) are SUB-CONCEPTUAL knowledge elements -- primitive read-outs of experience.
Alternative conceptions are HIGHER-LEVEL prediction patterns that may recruit multiple p-prims.
These are NOT synonyms.

## How This Project Should Use Terms
- "p-prim" = the knowledge element (force as mover, dying away, Ohm's p-prim, etc.)
- "alternative prediction" = what a student produces when certain p-prims are activated
- "p-prim activation" or "p-prim signature" = what inference reveals about parameter values
- AVOID: "extracting p-prims" (overstates what inference does)
- AVOID: "p-prim or alternative conception" as if synonymous

## Non-Standard P-Prim Labels Used in Project
- "Force as deflector" -- NOT a diSessa p-prim; it's the Newtonian understanding (coordination class level)
- "Bigger means slower" -- a prediction pattern, not a named p-prim
- "Closer means stronger" -- a prediction pattern, not a named p-prim
- "Straight then curve" -- a trajectory pattern, not a p-prim
- "Continuing push" -- close to diSessa's "continuous force" but the description is wrong (says "keep accelerating" but the intuition is about maintaining velocity, not increasing it)

## Standard diSessa P-Prims Relevant to This Project
- Force as mover (diSessa 1993): force sets velocity directly
- Dying away: effects diminish without sustained cause
- Ohm's p-prim: more effort/cause -> more effect
- Continuous force: sustained motion requires sustained cause
- Balancing: systems tend toward equilibrium
- Maintaining agency: effects require maintained causes
